地理院地図の3D機能を使えば、地形データの3Dモデルをダウンロードできる。
これを用いて山を3Dプリントしてみた。
![Pasted image 20260101120115.jpg|500](/img/user/_config/Extra/Pasted%20image%2020260101120115.jpg)


### 3Dモデル出力
[国土地理院地図](https://maps.gsi.go.jp)で、上のメニューの「ツール」から「3D」を使うと、任意の範囲の3Dモデルを作成できる。
![Pasted image 20251229111857.png](/img/user/_config/Extra/Pasted%20image%2020251229111857.png)

「大」「小」「カスタム」があるが、「大」「小」だと範囲を指定しづらい。
そこで今回は「カスタム」を使う。
まずは「大きさを固定」モードで3Dモデル化したい対象を範囲内に収める。ついでに、大きさの縦横を揃えておく。
![Pasted image 20251229112255.png](/img/user/_config/Extra/Pasted%20image%2020251229112255.png)

次に、「範囲を指定」に切り替えて、希望の縮尺になるようマウスホイールでズームする。今回は、一回だけズームインしてこれくらいの縮尺にした（3D範囲のサイズは512x512から1024x1024になった）。

![Pasted image 20251229112537.png](/img/user/_config/Extra/Pasted%20image%2020251229112537.png)

「OK」を押すと、3Dモデルが生成される。ここからはBlenderで作業するため、3Dモデルを出力する。ファイル形式はいくつかあるが、ここでは「STLファイル」をダウンロードする。

![Pasted image 20251229112743.png](/img/user/_config/Extra/Pasted%20image%2020251229112743.png)

![Pasted image 20251229113147.png](/img/user/_config/Extra/Pasted%20image%2020251229113147.png)

### Blenderに取り込んで
ダウンロードしたSTLファイルをそのままスライサーソフトに入れてもいいが、少し加工したいのでBlenderで取り込む。

ダウンロードしたdem.stlをBlenderにインポートする。
いつからかは分からないが、ドラッグ&ドロップでインポートできるようになってて便利。
インポート時のオプションで、Forward AxisをX、Up AxisをYにしてインポートする（インポート後に回転させてもよいが）。

![Pasted image 20251229113440.png](/img/user/_config/Extra/Pasted%20image%2020251229113440.png)

山の模型だと凹凸をわかりやすくするために高さ方向の縮尺だけ大きくする場合もあるが、今回はそのままで。
![Pasted image 20251229113719.png](/img/user/_config/Extra/Pasted%20image%2020251229113719.png)

せっかくなので、山の名前と標高のプレートを作った。

![Pasted image 20251229125842.png](/img/user/_config/Extra/Pasted%20image%2020251229125842.png)

stlでエクスポートして、bambu studioに取り込む。特別な設定はせずにスライス。
![Pasted image 20260101144836.png](/img/user/_config/Extra/Pasted%20image%2020260101144836.png)





山名プレートつける向き間違えた（手前に南が来るようにしたかったが、逆にしてしまった）


補足
山を3Dプリントするなら、BasicよりもMatteのほうが良さそう。光の反射で見づらくなる角度がある。

冬になって気温が低くなってきたからか、収縮によって端のほうが浮き上がっておりヒヤヒヤした。そういう点ではエンクロージャ付きのほうが安定するのかね。
